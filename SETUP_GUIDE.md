# دليل الإعداد والتشغيل السريع (Mock Checkers)

**ملاحظة هامة:** تم استبدال جميع بوابات الفحص الحقيقية (Checkers) بنسخ وهمية (Mock Checkers) لغرض الاختبار. هذه البوابات الوهمية تعطي نتائج عشوائية (Live, CCN, Dead) ولا تقوم بإجراء أي معاملات حقيقية.

## 🚀 خطوات التشغيل

### الخطوة 1: تثبيت المتطلبات

```bash
pip3 install -r requirements.txt
```

أو تثبيت يدوي:
```bash
pip3 install aiogram==2.25.1
pip3 install aiohttp==3.9.1
pip3 install aiofiles==23.2.1
pip3 install telethon==1.34.0
pip3 install pyjwt==2.8.0
pip3 install cryptography==41.0.7
```

---

### الخطوة 2: تحديث الإعدادات

افتح ملف `config.py` وقم بتحديث:

```python
API_TOKEN = 'YOUR_BOT_TOKEN_HERE'  # توكن البوت من @BotFather
Admin = YOUR_ADMIN_ID              # معرف التيليجرام الخاص بك
auth_chats = {
    'private': [YOUR_ADMIN_ID],
    'group': [-1002773091911]      # معرف المجموعة المصرح بها
}
```

**كيفية الحصول على معرفك:**
1. أرسل `/start` لـ @userinfobot
2. سيعطيك `Id: 123456789`
3. استخدم هذا الرقم في `Admin`

---

### الخطوة 3: إعداد premium.json

تأكد من وجود ملف `premium.json` بالشكل التالي:

```json
{
  "premium_users": []
}
```

إذا لم يكن موجوداً، سيتم إنشاؤه تلقائياً عند أول تشغيل.

---

### الخطوة 4: تشغيل البوت

```bash
python3 main.py
```

إذا ظهرت رسالة:
```
INFO:aiogram:Bot started
```
معناها البوت يعمل بنجاح! ✅

## 🧪 اختبار البوت (Mock Checkers)

**ملاحظة:** جميع أوامر الفحص ستستخدم الآن البوابات الوهمية وستخصم نقطة كريدت واحدة لكل فحص.

### 1. اختبار Admin
```
1. أرسل /start للبوت
2. أرسل /sup → يجب أن يظهر "ADMIN" مع "∞"
3. أرسل /b3 4532015112830366|12|2028|123
4. يجب أن يعمل بدون خصم ويعطي نتيجة عشوائية (Mock)
```

### 2. إضافة مستخدم بريميوم
```
1. أرسل /admin
2. اختر "💰 Add Credits"
3. أدخل: 10
4. أدخل معرف المستخدم
5. ✅ تم إضافة 10 نقاط
```

### 3. اختبار الفحص
```
1. المستخدم البريميوم يرسل /sup
2. يجب أن يظهر "PREMIUM" مع "10"
3. يرسل /b3 4532015112830366|12|2028|123
4. الكريدت يصبح 9 ويعطي نتيجة عشوائية (Mock)
```

## 📋 الأوامر الأساسية

### للمستخدمين
- `/start` - بدء البوت
- `/sup` - عرض معلومات الاشتراك
- `/cmd` - عرض قائمة الأوامر
- `/b3 CC|MM|YY|CVV` - فحص بطاقة (Mock)
- `/mass` - فحص جماعي (Mock)

### للإدارة
- `/admin` - لوحة الإدارة
  - إضافة كريدت
  - تفعيل Unlimited
  - عرض المستخدمين
  - حذف مستخدم

## ⚙️ الإعدادات المتقدمة

### تغيير مهلة المستخدمين المجانيين

في `config.py`، السطر 39:
```python
if key in last_use and current_time - last_use[key] < 30:  # 30 ثانية
```

غيّر `30` إلى المدة المطلوبة بالثواني.

### إضافة مجموعات مصرح بها

في `config.py`:
```python
auth_chats = {
    'private': [608548316],
    'group': [-1002773091911, -1001234567890]  # أضف معرفات المجموعات
}
```

**كيفية الحصول على معرف المجموعة:**
1. أضف @userinfobot للمجموعة
2. أرسل أي رسالة
3. سيعطيك معرف المجموعة

## 🔧 حل المشاكل الشائعة

### المشكلة: `ModuleNotFoundError: No module named 'aiogram'`
**الحل:**
```bash
pip3 install aiogram==2.25.1
```

### المشكلة: `FileNotFoundError: premium.json`
**الحل:**
```bash
echo '{"premium_users": []}' > premium.json
```

### المشكلة: البوت لا يستجيب
**الحل:**
1. تأكد من صحة التوكن في `config.py`
2. تأكد من أن البوت يعمل: `ps aux | grep python`
3. تحقق من السجلات (logs) في Terminal

### المشكلة: `Please wait 30 seconds`
**الحل:**
- هذا طبيعي للمستخدمين المجانيين
- أضف المستخدم كبريميوم عبر `/admin`

## 📊 مراقبة البوت

### عرض السجلات
```bash
python3 main.py 2>&1 | tee bot.log
```

```bash
nohup python3 main.py > bot.log 2>&1 &
```

```bash
pkill -f main.py
```
